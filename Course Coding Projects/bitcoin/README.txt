#### Bitcoin Price

This project is for getting the price of bitcoin from coindesk.com and will take the user's input from a command-line input to decide how much x bitcoin is.

## digit_check
Will confirm that the user put in a number.

## get_total
Will use requests.get("api link") and response.json() to get the information and format it into json. The user input will be multiplied by the information retrieved from the json.
