### Guessing Game

This project is to showcase interacting with the user through a guessing game. The program will us a random number generator based on the "Level" that is selected by the user. 

## main

This is the only function and ask for a user input which will be the highest number in the random range will generate. Will let the user know if their guess is too low, too high, or jsut right. If right qill quit the program.
